一点都不正经的互动问答节目

和你以往看到的 ASK / Q&A 节目本质上一样，但它就是没那么正经

参与提问 | 限制内容

参与《不正经问答》则视为您已阅读并同意 限制内容与须知 和 标记系统 中的内容。


（也许不是很可靠的）公告

由于个人精力与状态，以及提问内容（或者参与者）存在问题，第 5 期《不正经问答》将成为《不正经问答》的最后一期节目。此后，《不正经问答》将被“停播”。不过，在第 5 期《不正经问答》发布后，我们仍将继续接收评论。敬请期待“最后一期”《不正经问答》。

这是什么？

《不正经问答》是由 CarrotTV 出品的一档将整活进行到底的互动问答类节目。该节目没有固定的更新频率，基本上是随缘更新。这种过气节目能播出第 0 期的时候，我都觉得抽象了。

这种 ASK / Q&A 类型的节目玩法都差不多，都是在评论区发出你的提问，然后你会获得解答。
然而，在《不正经问答》中，你获得的解答基本上不可能是正经的，包括但不限于：

  • 无情的 AI 语音
  • 已读乱回（这是真的，本节目包含大量已读乱回时刻，请做好被已读乱回的准备）
  • 梗小鬼
  • “关音菩萨”警告 / 忽然惊吓
  • CarrotTV 邀您欣赏 15 秒广告，每秒 6 个广告（《不正经问答》第 1 期）
  • 你是禽兽吧，CarrotTV？（VDS《Q&A Time》第 6 期中对 CarrotTV 行为艺术提问的吐槽）
  • 看不懂的提问被丢给 豆包大模型 解释，然后直接偏离提问原始意图
  • 充电解锁不带变声器的行为艺术演绎（我错了我再也不整这种烂活了）
  • 忽然被 @CarrotTV 回复
  • 未来莫名其妙改版可能

总之就是烂活。

我也想来玩

怎样才能加入《不正经问答》的提问阵容呢？

  1. 我们进入《不正经问答》的 最新一期 的视频页面，
  2. 找到评论区，
  3. 确定置顶评论没有说“暂停接收提问”，或者评论区没有处于“精选评论”模式，
  4. 阅读《不正经问答》的 限制内容 部分，确认你的提问不属于被限制的内容，
  5. 直接在评论区输入你想发表的问题，发送评论后就可以加入进来了。
    注：评论开头加不加“ask:”前缀取决于你自己，可加可不加。
    但我可以明确告诉你，有一部分用户单纯不喜欢“ask:”前缀。放心，我绝对不会因为这个而骂你。而且《不正经问答》现已限制“秀优越感”的提问。

请注意！参与《不正经问答》则视为您已阅读并同意 限制内容与须知 和 标记系统 中的内容。

同时，请提问者尽可能确保 语言的基本通顺。如果我无法理解你提问的内容，你会被 标记 并被 降低回答优先度，同时我将会把无法理解的提问丢给 豆包大模型 解释，这可能导致最终的回答不符合提问者的想法。

以及如果你在往期（已开启“精选评论”）的评论区中提问，那么相应提问多半不会被收录到下一期节目中。请务必在最新一期《不正经问答》的评论区中发表提问。

同时，请您知悉，《不正经问答》虽然是一档很水的互动问答节目，但是它并不能无限度地接受来自同一参与者（包括小号）的大量提问。请参与者将单期的同时提问数量控制在合理范围内，严格意义上，单个参与者在单期节目 不得提出超过 12 个问题。我们不希望同一提问者在单期节目中发布过多提问，这样只会增添这档低成本整活节目的麻烦。

此外，我们可能无法保证您在本节目中呈现的用户昵称与头像能做到实时更新，换句话说，节目上线时，可能您会看到自己的头像和昵称还停留在自己的旧头像 / 昵称。对于此类事件在此致歉。
同时，参与者频繁更换头像与昵称的行为也可能会影响《不正经问答》的制作。我们可能会因为过于频繁的昵称 / 头像更改而放弃及时跟进相关修改。（换句话说，节目中展示的头像 / 昵称依旧是旧头像 / 昵称，对于过于频繁的信息修改，我们无力跟进）

所以谁能推荐一下修复赛博失语症的地方。
以及应该没人看出来我这段文案在效仿科技美学吧。

关于标记系统

《不正经问答》自认为是一档足够包容的问答节目，得益于 CarrotTV 的 B 站账号有极为宽松的黑名单机制（详见 carrottv.com.cn/2030），基本上绝大多数用户都可以参与本节目的提问。
*不幸的是，其实除了 CarrotTV 账号黑名单之外，《不正经问答》也已经限制了某部分用户的参与。但它不会体现在账号黑名单中。

然！而！出于各种奇奇怪怪古里八怪我也不知道怎么解释 就算解释了也不能播 反正就是很奇怪的原因，这类问答类节目（包括下方推荐的节目）或多或少都会被一些难以理解的提问，以及一些确实很无理的提问者（我不是说全部，这里的“无理”指的是“无理取闹”，而不是“提问无法理解”，别乱对号入座）占据。

即便如此，《不正经问答》还是决定，只通过“语言的基本通顺”这一个要素 来对一些提问者进行标记。也就是说，只要你的提问符合一般逻辑，能被人看懂（这里的“看懂”指的是可读，也就是说，玩梗也可以，我会尽力去找梗的），那《不正经问答》就不会标记你的提问。

被标记的后果目前只有一个，就是相应提问会被我拿去用 豆包大模型 解释。至于最后回答的思路符不符合想法，那我真不知道。应该大概是符合的。

好像“标记”系统还是《不正经问答》首创的。

注：标记系统与限制内容系统为并行关系。
注 2：标记系统的意义其实是整活，但我们依旧提出以下请求：

  1. 如果参与者无法对自己的行为做出正确认知（或者说参与者完全不知道自己在干什么），请谨慎参与《不正经问答》。
  2. 如果用户已表现出智力残疾症状（包括但不限于就读于启智学校 / 与治疗智力残疾有关的特殊学校、有智力残疾相关的残疾证），请在深思熟虑后再参与《不正经问答》。建议尽早寻找康复途径。

限制内容与须知

在参与《不正经问答》提问时，请注意，我们不接受以下类型，或涉及下述内容的提问。这些类型的提问均会被忽略。严重违反相应规定者可能会被禁止参与《不正经问答》以及 CarrotTV 账号的后续互动。

  • 违反 bilibili 社区规范 的提问
  • 透露自己或他人隐私的提问，以及试图直接索取或以威胁方式索取或透露 CarrotTV 与其运营人员相关隐私的提问
  • 纯粹复制其他用户提问（可能仅仅只是更改了几个字符）的提问
  • 以“挂人”、人身攻击为目的的提问
  • 请求他人“解除黑名单”的提问
    注：此处的“他人”包括 @自言者一么 和 @CarrotTV。
  • 与 Outfit7 旗下《汤姆猫和他的朋友们》IP 相关的提问
    注：相关游戏玩家群体过于低龄，以及 CarrotTV 的创建者经历了意外丢失《我的安吉拉》等游戏云存档的情况。
  • 与 CBeebies 的 Numberblocks（数字方块）及其同类 IP 有关的提问
    注：你可以试试 在 B 站搜索“数字方块”，然后再来想想这个东西的“二创圈子”到底适不适合小孩。
  • 与“天文圈”（虚构星球相关圈子）、“星球动画”有关的提问
    注：这些圈子不一定是糟糕的，但我方已经被“天文圈”中的某些用户恶心到了。这是无奈之举。
  • 点歌时点播 Windows XP 原版开关机音乐
    注：仅仅只是因为 CarrotTV 的创建者害怕 Windows XP 原版开关机音乐。使用此音效制作的“二次创作”曲目不受此限制。
  • 点歌时点播儿歌
    注：《不正经问答》并不适合儿童观看，我们不希望由此带来不必要的麻烦。《一年级》这首歌除外。
  • 纯粹复读当期节目内容的提问
  • 违背公共秩序的提问
  • 毫无语言逻辑，导致豆包等人工智能模型或者任何一个正常人 完全无法理解 的提问
    注：虽然该规定与标记系统存在一定冲突。具体来说，彻头彻尾无法让人理解 的提问才会被禁止。
  • 请求互关的提问
    注：你可以看看 CarrotTV 账号只关注了谁。
  • 表现出“优越感”的提问
  • （待补充，内容随时可能更新，恕不另行通知）

  • 除了 CarrotTV 账号黑名单之外，《不正经问答》也已经限制了某部分用户的参与，但它不会体现在账号黑名单中；我们有权无视来自这部分用户的提问。该部分的用户名单不会被公开。
  • 发表请求互关、互赞、组队完成活动的提问，或者发表（或用户信息包含）“代肝”相关信息，很有可能 会导致您被 CarrotTV 和 @自言者一么 的账号拉黑。

  • 读到我结束
    注：具体表现特征暂时没定好，但是绝对不会正经
  • 点歌
    注：单纯是因为我之前在 VDS 的《Q&A Time》那边干过这种事情
    注 2：涉及付费专辑的歌曲，以及在 限制内容 中涉及到的相关歌曲无法在节目中播出
    注 3：单期节目点歌数量不宜过多；从第 5 期节目开始，同一用户（含小号）在单期节目只能点播一首歌
    注 4：多余的点歌将会进入队列（队列机制仍在构想）
    注 5：我们可能会以惩罚、整活等理由对点歌进行“重定向”。

    注：后果自负。
